# Number Combination Generator (PWA)

This application is a simple **Progressive Web App (PWA)** that allows users to generate **random combinations of 5 numbers** from a custom list of numbers provided by the user.

Users can add as many numbers as they want to a list. Once the list is created, they simply need to click the **Generate** button to produce a random combination of **5 unique numbers** selected from that list.

The application is **optimized for mobile phones** with a simple interface, large buttons, and a layout adapted to small screens.

## Features

* Add custom numbers to a list
* Random generation of a **5-number combination without repetition**
* Display of the list of added numbers
* Ability to **completely reset the list**
* **Responsive interface optimized for smartphones**
* **Installable mobile application (PWA)**
* **Offline functionality thanks to the Service Worker**

## Technologies Used

* HTML
* CSS
* JavaScript
* Progressive Web App (Manifest + Service Worker)

## Installation

1. Clone the project from GitHub
2. Open the project in a browser or deploy it on a hosting service (e.g., Vercel)
3. On mobile, use the **Add to Home Screen** option to install the application

## Goal

This project was created to provide a simple tool that allows users to quickly generate number combinations from a custom list while offering an experience similar to a mobile application.
